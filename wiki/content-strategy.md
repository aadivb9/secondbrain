# Content Strategy: Instagram Reels & Short-Form Video

## The Hook Window

You have **1.7 seconds on mobile** to stop the scroll. Everything else is secondary. The hook lands in the first frame — visually, not just in audio.

## Hook Taxonomy

| Type | Example | Best for |
|------|---------|----------|
| Curiosity gap | "We need to talk about..." | AI/tech topics |
| Bold claim | "Everyone's doing this wrong." | Contrarian takes |
| Problem-solution | "Struggling with X? Here's the fix." | Tutorial content |
| Transformation | "How I went from 0 to..." | Personal brand |
| Targeting | "This one is for CS students." | Niche community |
| FOMO | "You need this tool." | Tool/resource drops |
| Insider reveal | "Here's what most people don't know." | Research/deep dives |

## Psychological Triggers (ranked by power)

1. **Curiosity** — incomplete loop forces completion
2. **Social belonging** — "this is for people like you"
3. **Authority/insider** — "secret", "reveal", behind-the-scenes
4. **FOMO** — urgency + scarcity language
5. **Emotional resonance** — story beats, relatable moments

## Structure

```
[0–2s]  Hook — bold text overlay or verbal pattern interrupt
[2–20s] Body — one insight, one story beat, one demo
[20–30s] CTA — "follow for more [X]" or "comment [Y]"
```

- Optimal length: **15–30 seconds** for Instagram Reels
- Caption everything — 50% watch on mute
- 9:16 vertical, safe zones for text (avoid platform UI overlap)
- Visual hook works even without audio

## Posting Strategy

- **Instagram Reels:** 3–5x/week minimum
- Quality 3x/week > low-effort daily
- 80/20 rule: 80% original, 20% trend-riding
- Batch create — film 3–5 in one session

## Winning Formats for CS/AI Brand

1. **Tool drop** — "This tool does X in 2 seconds" → demo → CTA
2. **Myth-bust** — "Everyone thinks X. Here's why they're wrong."
3. **Day-in-the-life** — authentic, behind-the-scenes
4. **Anime angle** — apply anime moment to CS/life lesson
5. **Quick tutorial** — one command or concept in 20 seconds

## Performance Stats

- Reels: 2× impressions vs other post types (Meta)
- 140B daily Reel plays
- Short-form = #1 ROI format in 2026 (HubSpot)

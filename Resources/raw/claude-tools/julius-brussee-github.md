## Key Repositories

### caveman (38,913 stars)
Token compression skill for Claude Code and other AI agents. Reduces output tokens 65–75% using terse "caveman grammar". Four levels: Lite, Full, Ultra, Classical Chinese. Related tools: `/caveman-commit`, `/caveman-review`, `/caveman-compress`. Works with Claude Code, Codex, Gemini CLI, Cursor, Windsurf, Cline, 40+ agents.

### revu-swift (187 stars)
Local-first macOS spaced repetition study app. FSRS scheduling, decks/flashcards/exams/study guides, Anki import (.apkg/.colpkg), CSV/Markdown import. SwiftUI, SQLite local storage, no cloud. Keyboard-optimized. MVVM architecture.

### cavekit (598 stars)
Claude Code plugin for spec-driven parallel development. Four commands: sketch → map → make → check. Parallel task execution with dependency tiers. Multi-model review with Codex. Requirement tracing (R-numbers). Circuit breakers. Team coordination via git-backed task claims.

### cavemem (33 stars)
Cross-agent persistent memory via MCP. Captures session events, compresses with caveman grammar (~75% reduction), stores in local SQLite. Three MCP search tools. Web viewer at localhost:37777. Privacy: strips `<private>` tags. Works with Claude Code, Cursor, Gemini CLI.
